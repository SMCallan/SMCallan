## Featured Architecture

### 🇬🇧 DevSecOps Market Intelligence Engine

An extensible, edge-computed market intelligence framework designed to aggregate, parse, and analyse unstructured job-board data at scale.

While currently configured to provide statistical analysis of the London cybersecurity market, the underlying engine is customisable and can be adapted for OSINT dashboards across different sectors, technology stacks, and geographic regions.

#### What it demonstrates

- **Configurable ingestion:** Daily GitHub Actions trigger a Python pipeline that collects targeted role data from Adzuna and Reed APIs.
- **Data cleaning and normalisation:** Regex-based processing handles noisy salary strings, inconsistent formats, duplicate records, and unstructured job-board data.
- **Edge analytics:** A Cloudflare Worker API performs on-the-fly analysis, including salary modelling and separation of contract day rates from permanent salaries.
- **Accessible frontend:** A Vite + React interface designed around clarity, visual chunking, low-friction search, and neurodivergent accessibility.
- **Edge deployment:** Search, filtering, pagination, and analytics are served from Cloudflare infrastructure for low-latency interaction.
